# 🎮 Neon Cube Runner - Quick Start

## 🚀 Play Instantly!
Open `index.html` in your browser and click **"▶ START GAME"**!

## 🎯 Controls
| Control | Action |
|---------|--------|
| Mouse | Move cursor to steer |
| ← → Arrows | Move left/right |
| A / D Keys | Move left/right |
| Spacebar | Pause/Resume |

## 🎮 Objective
1. **AVOID** 🔴 red cubes
2. **COLLECT** 💎 colorful orbs
3. **SURVIVE** as long as possible!

## 💎 Collectibles
- 🟡 Yellow: Sun (100 pts)
- 🟢 Green: Nature (100 pts)
- 🟣 Purple: Magic (100 pts)
- 🔵 Cyan: Energy (100 pts)

## ⚡ Quick Tips
1. Use Mouse - More precise
2. Focus Combos - Build multipliers
3. Look Ahead - Anticipate obstacles
4. Stay Centered - Max maneuvering

## 🔧 Troubleshooting
- Game won't load? Try Chrome/Edge/Firefox
- No sound? Game has no audio (by design)
- Lag? Close other tabs, use Chrome/Edge
